Plugin for browser Notifications API:

INSTALL:

1 - Install the plugin;

2 - In GLPI -> Configuration -> Plugins click in plugin name to go to config page.

3 - Enable the plugin.




UNINSTALL

1 - In GLPI -> Configuration -> Plugins click in plugin name to go to config page.

2 - Disable the plugin.

3 - Uninstall the plugin;
